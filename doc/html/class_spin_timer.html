◆ isExpired()

bool SpinTimer::isExpired ( )

Poll method to get the timer expire status, recalculates whether the timer has expired before. This method could be used in a pure polling mode, where tick() has not to get called (by the TimerContext::handleTick() method), but also a mixed operation in combination with calling tick() periodically is possible. Subsequent isExpired() queries will return false after the first one returned true, as long as the time did not expire again in case of a recurring timer.

Returns
true if the timer has expired.
